# java-calculator
문자열 계산기 미션 저장소

## 기능

- [x] 사용자 입력을 받는 기능

- [x] 사용자 입력을 숫자와 기호로 구분하는 기능

- [x] 사용자 입력 예외처리(정규표현식 사용해서 모든 예외처리함)
    - [x] 사용 가능한 기호("+, -, *, /, ~~.~~) 제외 입력 예외처리 기능
    - [x] 숫자나 기호가 연속으로 입력된 경우
    - [x] 공백만 입력된 경우 예외처리 기능
    - [x] 공백이 연속으로 입력된 경우 예외처리 기능
    - [x] 입력 숫자가 Double 범위를 벗어 난 경우 예외처리 기능
    - [x] 0으로 나누는 경우

- [x] 계산해주는 기능

- [x] 계산결과를 출력해주는 기능
